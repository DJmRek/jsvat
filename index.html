jsvat

Check the validity of the format of an EU VAT number specified

Download this project as a .zip file Download this project as a tar.gz file

Demo

The result is:
There are few valid codes:
  • ATU00000024
  • BE0414445663
  • RS100182160
  • RU0274051582
There are few invalid codes:
  • ATU10223001
  • BE0000200334
  • RO000099908
  • RU5027187067
Example of usage:
jsvat.checkVAT('ATU00000024');
